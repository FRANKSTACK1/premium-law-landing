# Sir Stanleys Law Firm — Premium Landing Page

A sophisticated, fully responsive landing page designed for a high-end legal practice. This project focuses on bridging the gap between traditional corporate prestige and modern web performance.

## ✨ Key Features

*   **Premium Aesthetic:** Clean, modular layout with a focus on typography and high-contrast imagery.
*   **Fully Mobile-Responsive:** Seamless transitions across mobile (320px+), tablet, and ultra-wide desktop views.
*   **Smooth Interactions:** Subtle fade-up scroll animations and hover effects for a "cinematic" feel.
*   **Optimized Header:** A sticky navigation bar with background blur (glassmorphism) and scroll-sensitive shrinking.
*   **Custom Mobile Menu:** A lightweight, accessible slide-in hamburger menu with backdrop blur.
*   **Performance First:** Built using Vanilla JavaScript to ensure zero dependency bloat and lightning-fast load times.

## 🛠️ Technical Stack

- **HTML5** & **Tailwind CSS** (Utility-first styling)
- **Vanilla JavaScript** (DOM manipulation & scroll logic)
- **Google Fonts** (Noto Serif for headings, Work Sans for body)
- **Material Symbols** (Minimalist iconography)

## 🚀 Improvements Made

As part of the **FRANKSTACK** optimization process, I implemented:
1. **Fluid Grid Systems:** Converted static containers into responsive flex/grid layouts.
2. **Accessibility (a11y):** Added ARIA labels and keyboard-friendly navigation.
3. **Scroll Logic:** Custom JavaScript for "Back to Top" visibility and scroll-reveal triggers.
4. **UX Enhancements:** Touch-friendly button targets and refined padding for mobile users.
